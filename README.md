
# 🎉 Interactive Birthday Wish Website

A beautifully animated, interactive birthday celebration web page featuring a keypad lock screen, floating balloons, a virtual cake-cutting experience, and a custom website generator.

---

## 🚀 How to Use

### 1. Accessing the Birthday Page
* **Open the Link:** Click the live GitHub Pages link provided in the repository settings.
* **Enter the Passcode:** On the lock screen, enter the **4-digit passcode** (Hint: Birthday Date in `DDMM` format, e.g., `1506` for June 15th).
* **The Shifting Icons:** Watch the live pop-culture icons (cartoons, cricketers, and Bollywood stars) cycle next to the passcode display every second!

### 2. The Celebration Experience
* **Rip the Wrapper:** Click the pink ribbon bow to "rip" the gift wrapping paper away.
* **Open the Gift:** Click the floating gift box to reveal the birthday cake module.
* **Cut the Cake:** Click the **"Cut the Cake 🎂"** button to trigger the virtual knife animation, extinguish the flame, and reveal the personalized wish banner.
* **View the Gallery:** Tap the floating candles (`🕯️🕯️🕯️`) to slide down to the Polaroid Photo Gallery and Surprise Checklist.

---

## 🛠️ Creating Your Own Custom Birthday Link

You can use the built-in **Creator Engine** at the bottom of the page to generate a personalized link for a different friend without changing any code:

1. Scroll to the footer and click **"Create your own birthday website for free ✨"**.
2. Fill out the configuration form:
   * **Friend's Name:** Enter their name to personalize the headers.
   * **Secret Passcode:** Set a custom 4-digit unlock code.
   * **Custom Message:** Type your personal birthday wish.
   * **Music Track:** Select a default track or paste a direct `.mp3` URL.
   * **Polaroid Images (Optional):** Paste image web URLs and captions to update the photo gallery.
3. Click **"Generate Website URL ✨"**.
4. **Copy the generated link** from the dashed box and send it to your friend!

## ⚙️ Technologies Used
* **HTML5** - Structured semantic layout and dynamic audio integration.
* **CSS3** - Custom keyframe animations, interactive hover scaling, and a fluid 4-way gradient background.
* **JavaScript (ES6)** - Keypad validation logic, interval-based icon cycling, and URL query parameter parsing for dynamic customization.
